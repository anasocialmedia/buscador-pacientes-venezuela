## Categorias de producto observadas

### Hogar / cocina
- Maquina selladora, veggie slicer, pelador electrico
- Estructura que funciona: problema cotidiano claro → solucion practica → beneficios concretos → CTA urgencia

### Textil / prendas
- Polerones polares (3 variantes)
- No se ha construido landing en esta sesion para esta categoria

### Tecnologia personal / gadgets
- Vaso Termo Digital (primer producto activo)
- Pistola de masaje de percusion (segunda landing construida)
- Angulo que aplica: frustracion con alternativas existentes → producto que lo resuelve en casa, sin friccion

### Cuidado personal / bienestar
- Irrigador bucal, filtro ducha, pistola de masaje, corrector de postura
- Regla clave: NUNCA afirmaciones medicas o terapeuticas. Enmarcar como "relajacion", "bienestar", "comodidad", "ayuda a", "favorece", "contribuye a".
- Para postura: el angulo "trabajador de oficina / teletrabajador" convierte bien — el dolor es cotidiano y reconocible.

### Calzado / accesorios
- Secador de zapatos

### Ergonomia / habito corporal
- Corrector de postura (primera landing de esta sub-categoria)
- Estructura optima: Problema reconocible (cuello caido, hombros encorvados) → Solucion discreta → Beneficios → Como funciona (3 pasos) → Para quien es → Comparacion vs. sin corrector → FAQ → CTA final
- Seccion "Para quien es" funciona bien para ampliar el publico sin diluir el angulo principal (oficina, estudio, teletrabajo, gaming)
- Seccion comparacion (con / sin corrector) aumenta percepcion de valor en esta categoria

**Why:** El operador testea productos variados. Tener mapeadas las categorias permite reutilizar angulos y estructuras sin partir de cero.

**How to apply:** Antes de construir una nueva landing, revisar si el producto cae en una categoria ya testeada y adaptar el angulo en lugar de inventar uno nuevo. Para productos de habito corporal o ergonomia, usar la estructura del corrector de postura como referencia.
