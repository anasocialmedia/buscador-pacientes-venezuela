## Categorias de producto observadas

### Hogar / cocina
- Maquina selladora, veggie slicer, pelador electrico
- Estructura que funciona: problema cotidiano claro → solucion practica → beneficios concretos → CTA urgencia

### Textil / prendas
- Polerones polares (3 variantes)
- No se ha construido landing en esta sesion para esta categoria

### Tecnologia personal / gadgets
- Vaso Termo Digital (primer producto activo)
- Pistola de masaje de percusion (segunda landing construida)
- Angulo que aplica: frustracion con alternativas existentes → producto que lo resuelve en casa, sin friccion

### Cuidado personal / bienestar
- Irrigador bucal, filtro ducha, pistola de masaje
- Regla clave: NUNCA afirmaciones medicas o terapeuticas. Enmarcar como "relajacion", "bienestar", "comodidad".

### Calzado / accesorios
- Secador de zapatos

**Why:** El operador testea productos variados. Tener mapeadas las categorias permite reutilizar angulos y estructuras sin partir de cero.

**How to apply:** Antes de construir una nueva landing, revisar si el producto cae en una categoria ya testeada y adaptar el angulo en lugar de inventar uno nuevo.
