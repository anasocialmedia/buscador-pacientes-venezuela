Pistola de Masaje Percusiva — primer producto con copy generado (Mayo 2026).

**Ángulo principal seleccionado para cold traffic:** Tensión muscular acumulada por jornada laboral (cuello, espalda, hombros) → alivio inmediato en casa, sin masajista ni cita.

**Por qué:** Es el dolor más universal del segmento objetivo (trabajadores jornada completa) y coincide 1:1 con el ángulo de la landing. Maximiza coherencia anuncio→landing→conversión.

**Framework recomendado para ángulo principal:** PAS — el dolor es cotidiano, específico y fácil de reconocer en los primeros 3 segundos.

**Specs clave del producto a respetar en copy:**
- 30 niveles de velocidad, 3.300 RPM
- Batería 2.500 mAh (8 horas de uso), carga USB
- Precio venta: $35.990 CLP | Precio tachado: $49.990 CLP | Ahorro: $14.000 (precio confirmado y activo)
- COD, envío gratis a todo Chile

**Restricciones aplicadas:**
- No mencionar beneficios médicos ni terapéuticos garantizados
- No afirmar "elimina el dolor" — usar "alivia tensión", "relaja los músculos"
- Emojis: máximo 3-4 por pieza

**Notion Pipeline:** Actualizado a "✍️ Copy / Anuncios" el 2026-05-28.

**Why:** Primer producto con pipeline completo (landing + copy) — sirve de referencia para futuros productos de cuidado personal.

**How to apply:** Para productos similares de cuidado corporal/alivio de tensión, PAS con ángulo laboral (cuello/espalda) es punto de partida validado. BAB funciona bien como segundo ángulo para audiencias que ya reconocen el problema.
