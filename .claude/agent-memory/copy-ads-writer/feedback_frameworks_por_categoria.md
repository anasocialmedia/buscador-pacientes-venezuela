Patrones de frameworks por tipo de producto COD Chile (actualizados a medida que se generan campañas):

**Cuidado corporal / alivio de tensión muscular (ej. Pistola de Masaje):**
- PAS es el framework más directo: el dolor es cotidiano (cuello, espalda, hombros por trabajo) y universalmente reconocible en 3 segundos.
- BAB funciona como segundo ángulo para contraste "antes/después" de la rutina.
- AIDA para hooks de video donde la pregunta disruptiva ("¿Cuándo fue la última vez que...?") detiene el scroll.

**Hogar / cocina (ej. Veggie Slicer, Selladora Vacío):**
- PAS funciona bien con frustración cotidiana (comida que se echa a perder, cortar toma mucho tiempo).
- AIDA para hooks de demostración visual rápida.

**Textiles / prendas (ej. Polerones Polar):**
- BAB es el más natural: contraste frío/calor, antes sin la prenda / después con ella.
- El ángulo de regalo funciona bien como segundo ángulo en temporada.

**Postura / ergonomía (ej. Corrector de Postura Smart):**
- PAS es el framework principal: el dolor es acumulado e invisible (hombros, cuello por pantalla) — muy reconocible para teletrabajadores y universitarios.
- BAB funciona como segundo ángulo: contraste "sin corrector (encorvado, sin disciplina) / con corrector (postura mantenida sin esfuerzo)".
- Hook de interrupción de patrón ("Llevas 8 horas así y no te has dado cuenta") funciona bien como tercer ángulo de curiosidad.
- Diferenciador clave en categoría con 6+ competidores: especificidad del contexto (teletrabajo/pantalla) y discreción del producto (bajo la ropa).
- Competidores usan "Corrige Postura" directamente — PM se diferencia con lenguaje honesto: "ayuda a mantener", "te recuerda", menor riesgo regulatorio.

**Why:** Estos patrones emergen del catálogo histórico de Punto Mercado y de la naturaleza visual/emocional de cada categoría.

**How to apply:** Al recibir un nuevo producto, identificar la categoría y aplicar el framework correspondiente como primera opción. Documentar si el testeo real valida o contradice este patrón.
